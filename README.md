# On Deck

### UP & RUNNING
* `npm install`
* `npm start`
* visit `http://localhost:5000/`

### About
**v.1.0.0**
On Deck is hosted on heroku [here](https://on-deck.herokuapp.com/#/)

On Deck is a Webpack 3, React, Redux, and Sass application that keeps inventory.
